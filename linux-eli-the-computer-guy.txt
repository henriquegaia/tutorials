		6-Users, Groups and Permissions in Linux
		https://www.youtube.com/watch?v=zRw0SKaXSfI&list=PLD6B6473ACF32C59D&index=6
		practise from: --
				
			-flash drive mount/unmount
				sudo mkdir /mnt/drive
				fdisk -l
				sudo mount /dev/sdb1 /mnt/drive
				ls -l /mnt/drive
				sudo umount /dev/sdb1 /mnt/drive
				ls -l /mnt/drive

			-users 00:00 - 12:00
			
				-create user
					sudo adduser user1
				-change password
					sudo passwd user1
				-check users account and edit
					sudo vi /etc/passwd
				-remove user
					sudo userdel user1
					
			-groups 12:00 - 20:40
			
				-create group
					sudo groupadd grp1
				-delete group
					sudo groupdel grp1 
				-add user to group
					sudo adduser user1 grp1
				-delete user from group
					sudo deluser user1 grp1
				-edit groups 
					sudo vi /etc/group
					
			-permissions
				
				syntax: sudo chmod 	[ugoa] [+-=] [rwx],
									[ugoa] [+-=] [rwx], ... <file>
						sudo chmod 	<0-7><0-7><0-7> 		<file>
				
				-example for folder:
					sudo chmod xxx <dir> -R
					
			-ownership 30:00
			
				-sudo chown 	<user> <file>
				-sudo chown -R 	<user> <folder>
				
				-sudo chgrp 	<grp> <file>
				-sudo chgrp -R 	<grp> <folder>

				sudo service network-manager restart
				
			-interfaces conf file
				-sudo vim /etc/network/interfaces
				
			-setting the ip of an interface as ... 
			
				-dynamic
					-sudo vim /etc/network/interfaces
					
					auto eth0
					iface eth0 inet dhcp
					
					-restart networking service
					
				-static
					-sudo vim /etc/network/interfaces
					
					auto eth0
					iface eth0 inet static
					address 10.1.10.54 
					netmask 255.255.255.0
					network 10.1.10.0 
					broadcast 10.1.10.255 
					gateway 10.1.10.1
					
					-restart networking service
